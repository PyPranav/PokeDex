# PokeDex
PokeDex site made with Django
  
## Hosted Site:
https://pypranav.pythonanywhere.com/pokedex/ 
## Installation  
Use git clone to install it on local machine.
```bash
git clone https://github.com/PyPranav/PokeDex
pip install -r requirement.txt
```
